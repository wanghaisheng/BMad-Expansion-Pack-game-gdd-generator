# ------------------------------------------------------------
# Web Game Design Checklist
# ------------------------------------------------------------
---
checklist:
  id: web-game-checklist
  name: Web Game Design Checklist
  description: Ensure game design is optimized for web browsers and online distribution.
items:
  - "[ ] Game loads quickly in web browsers"
  - "[ ] Cross-browser compatibility verified (Chrome, Firefox, Safari, Edge)"
  - "[ ] Responsive design works on various screen sizes"
  - "[ ] Game works without plugins or additional downloads"
  - "[ ] Network connectivity requirements are reasonable"
  - "[ ] Offline functionality implemented where possible"
  - "[ ] Web-specific input methods supported (keyboard, mouse, touch)"
  - "[ ] Browser security restrictions respected"
  - "[ ] SEO optimization for game discovery"
  - "[ ] Social sharing and viral mechanics integrated"
  - "[ ] Progressive loading and streaming implemented"
  - "[ ] Memory usage optimized for browser limitations"
  - "[ ] Web analytics and tracking implemented"
  - "[ ] GDPR and privacy compliance addressed"
  - "[ ] Web monetization strategies integrated"
...

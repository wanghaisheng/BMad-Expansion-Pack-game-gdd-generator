# ------------------------------------------------------------
# Mobile Game Design Checklist
# ------------------------------------------------------------
---
checklist:
  id: mobile-game-checklist
  name: Mobile Game Design Checklist
  description: Ensure game design is optimized for mobile platforms and touch-based interactions.
items:
  - "[ ] Touch controls are intuitive and responsive"
  - "[ ] UI elements are appropriately sized for finger interaction"
  - "[ ] Game supports both portrait and landscape orientations (if applicable)"
  - "[ ] Performance targets meet mobile device limitations"
  - "[ ] Battery consumption is optimized"
  - "[ ] Game handles interruptions (calls, notifications) gracefully"
  - "[ ] Loading times are minimized for mobile networks"
  - "[ ] Offline functionality implemented where appropriate"
  - "[ ] App store guidelines compliance verified"
  - "[ ] In-app purchase integration follows platform standards"
  - "[ ] Push notification strategy is player-friendly"
  - "[ ] Game supports various screen sizes and resolutions"
  - "[ ] Accessibility features for mobile users implemented"
  - "[ ] Data usage is reasonable for mobile networks"
  - "[ ] Social sharing and viral features integrated"
...

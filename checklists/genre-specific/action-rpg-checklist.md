# ------------------------------------------------------------
# Action RPG Design Checklist
# ------------------------------------------------------------
---
checklist:
  id: action-rpg-checklist
  name: Action RPG Design Checklist
  description: Ensure Action RPG design balances combat, progression, and narrative elements effectively.
items:
  - "[ ] Real-time combat feels responsive and satisfying"
  - "[ ] Character progression provides meaningful choices"
  - "[ ] Skill trees offer diverse build possibilities"
  - "[ ] Equipment system provides clear upgrade paths"
  - "[ ] Loot drops are balanced and rewarding"
  - "[ ] Boss encounters are challenging but fair"
  - "[ ] Difficulty curve scales appropriately with player progression"
  - "[ ] Story integration doesn't interrupt gameplay flow"
  - "[ ] Character customization options are meaningful"
  - "[ ] Combat mechanics have depth for mastery"
  - "[ ] Resource management (health, mana, stamina) is balanced"
  - "[ ] Enemy variety provides different tactical challenges"
  - "[ ] Exploration rewards are worthwhile"
  - "[ ] Endgame content provides long-term engagement"
  - "[ ] Multiplayer elements (if any) are well-integrated"
...

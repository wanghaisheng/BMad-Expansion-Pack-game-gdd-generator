# ------------------------------------------------------------
# Simulation Game Design Checklist
# ------------------------------------------------------------
---
checklist:
  id: simulation-game-checklist
  name: Simulation Game Design Checklist
  description: Ensure simulation games provide realistic, deep, and engaging system interactions.
items:
  - "[ ] Simulated systems behave realistically and consistently"
  - "[ ] Complexity is introduced gradually to avoid overwhelming players"
  - "[ ] Cause-and-effect relationships are clear and logical"
  - "[ ] Data visualization helps players understand system states"
  - "[ ] Sandbox mode allows for creative experimentation"
  - "[ ] Goal-oriented scenarios provide structured challenges"
  - "[ ] System interactions create emergent gameplay"
  - "[ ] Performance remains stable with complex simulations"
  - "[ ] Save/load functionality preserves all system states"
  - "[ ] Modding support enables community content (if applicable)"
  - "[ ] Educational value is balanced with entertainment"
  - "[ ] Failure recovery mechanisms prevent total loss"
  - "[ ] Long-term engagement through evolving challenges"
  - "[ ] Accessibility features help diverse players engage"
  - "[ ] Documentation explains complex systems clearly"
...

checklist:
  id: technical-feasibility-checklist
  name: Technical Feasibility Checklist
  description: Ensure game design is technically achievable within project constraints.
items:
  - "[ ] Performance requirements realistic for target platforms"
  - "[ ] Asset complexity appropriate for team capabilities"
  - "[ ] Technical dependencies clearly identified"
  - "[ ] Platform-specific limitations considered"
  - "[ ] Development timeline realistic for scope"
  - "[ ] Required tools and technologies available"
  - "[ ] Memory and storage requirements within limits"
  - "[ ] Network/multiplayer requirements feasible"
  - "[ ] Localization requirements technically supported"
  - "[ ] Quality assurance approach defined and achievable"
  - "[ ] Post-launch update strategy technically viable"
  - "[ ] Asset naming rules and directory structure are enforceable"
  - "[ ] Asset metadata schema supports packaging filters and audits"
  - "[ ] Media profiles defined for video/audio/image_sequence/sprite_sheet"
  - "[ ] Automated validation gates defined (naming/metadata/media compliance)"
...

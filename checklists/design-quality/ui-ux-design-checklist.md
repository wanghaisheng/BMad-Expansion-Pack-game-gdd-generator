# ------------------------------------------------------------
# UI/UX Design Checklist
# ------------------------------------------------------------
---
checklist:
  id: ui-ux-design-checklist
  name: UI/UX Design Checklist
  description: Ensure user interface and experience design is intuitive, accessible, and enhances gameplay.
items:
  - "[ ] Navigation is intuitive and consistent throughout the game"
  - "[ ] Visual hierarchy guides player attention effectively"
  - "[ ] Interactive elements are clearly distinguishable"
  - "[ ] Feedback is immediate and appropriate for all interactions"
  - "[ ] Color scheme is accessible to colorblind players"
  - "[ ] Text is readable across all target devices and screen sizes"
  - "[ ] Loading states and progress indicators are informative"
  - "[ ] Error messages are helpful and actionable"
  - "[ ] Onboarding flow introduces UI elements progressively"
  - "[ ] Paywall and purchase flows are mapped with UX safeguards"
  - "[ ] Settings and customization options are comprehensive"
  - "[ ] Responsive design works across different screen sizes"
  - "[ ] Animation and transitions enhance rather than distract"
  - "[ ] Information density is appropriate for the context"
  - "[ ] Accessibility features are integrated seamlessly"
  - "[ ] UI performance doesn't impact gameplay performance"
  - "[ ] Screen inventory includes states, empty/error cases, and analytics events"
  - "[ ] UI storyboard covers key paths with triggers and localization keys"
  - "[ ] UI asset list includes icons, fonts, motion assets, and audio cues"
  - "[ ] UI assets specify format choice and performance budgets"
  - "[ ] UX philosophy is articulated and informs UI decisions"
...

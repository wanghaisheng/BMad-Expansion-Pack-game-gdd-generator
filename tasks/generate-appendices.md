# Generate Appendices

## Task Overview
创建 GDD 第 13 章「附录与资产清单」，包含资产索引、元数据、编码规范与参考文献。

## Objective
提供完整的资产与参考索引，用于生产与集成自动化校验。

## Prerequisites
- 资产清单与命名规范
- 元数据模式与校验要求
- 技术与艺术参考标准

## Inputs Required
- 资产列表（名称、类型、路径、版本）
- 元数据（schema、示例、校验结果）
- 编码与格式规范链接

## Agent Workflow

### Primary Agent: technical-designer
**Command**: `compile-appendices`
**Duration**: 60-90 分钟

#### Process Steps:
1. 汇总资产清单与分类（图片/音频/3D/特效/UI 等）
2. 附加元数据模式与示例文件位置
3. 列出编码与格式规范、工具指南与参考
4. 提供校验流程与质量门禁

## Deliverables
### Output: 13_Appendices.md
包含资产索引、元数据模式链接、格式与编码规范、参考资料。

## Success Criteria
- [ ] 资产索引完整且可检索
- [ ] 元数据模式与示例齐备
- [ ] 编码与格式规范清晰
- [ ] 校验流程与结果可追踪

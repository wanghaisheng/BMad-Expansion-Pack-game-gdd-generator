# Generate Management and Operations Plan

## Task Overview
创建 GDD 第 12 章「项目管理与运营规划」，覆盖里程碑、资源、风险、发布与运营策略。

## Objective
将设计落地到可执行的项目与运营计划，支持上线与持续服务。

## Prerequisites
- 技术与内容里程碑初稿
- 团队资源与预算约束
- 目标平台发布节奏与合规要求

## Inputs Required
- 里程碑拆分与时间线
- 角色与人力配置
- 风险清单与缓解策略
- 运营活动与内容更新计划

## Agent Workflow

### Primary Agent: pm-designer
**Command**: `create-management-plan`
**Duration**: 60-90 分钟

#### Process Steps:
1. 设定里程碑与交付标准
2. 资源与预算分配
3. 风险识别与缓解方案
4. 发布流程与平台合规
5. 运营活动与内容节奏

## Deliverables
### Output: 12_Management.md
包含项目计划、资源配置、风险管理、发布与运营策略。

## Success Criteria
- [ ] 里程碑明确且可执行
- [ ] 资源与预算匹配目标
- [ ] 风险与缓解方案完整
- [ ] 运营与更新节奏清晰

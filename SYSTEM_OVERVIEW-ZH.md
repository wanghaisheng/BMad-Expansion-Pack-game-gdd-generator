# Game Design Document Generator Team - System Overview

## 🎮 系统概述

这是一个完整的游戏设计文档生成系统，专门为创建高质量、标准化的游戏设计文档(GDD)而设计，并与BMAD游戏开发工作流程无缝集成。

## 📊 系统规模

### 核心组件统计
- **30个专业代理** - 涵盖游戏设计的所有专业领域
- **55个专业任务** - 详细的游戏设计执行指南
- **22个专业模板** - 全面的文档框架和规格
- **34个质量检查清单** - 确保各领域的质量标准
- **27个工作流程** - 针对不同场景的设计流程
- **9个知识库文件** - 支持设计决策的知识库

## 🎯 使用场景

### 新游戏开发
1. 使用**综合GDD创建**工作流程
2. 选择适当的平台和类型专家
3. 应用相关的质量检查清单
4. 生成标准化的GDD文档

### 快速原型验证
1. 使用**快速原型GDD**工作流程
2. 专注核心机制设计
3. 快速迭代和验证
4. 渐进式完善文档

### 跨文化市场扩展
1. 使用**跨文化GDD**工作流程
2. 应用文化适应模板
3. 执行文化敏感性检查
4. 制定本地化策略

### 特定平台优化
1. 选择平台特定工作流程
2. 应用平台专家指导
3. 执行平台兼容性检查
4. 优化平台特性利用

# Game Design Document (GDD) Generator

An AI-orchestrated generator that produces standardized, structured Game Design Documents (GDDs) aligned with BMAD game development workflows.

## 🎮 Overview

This repository is a GDD generator: it coordinates multiple AI capabilities (specialized agents, tasks, templates, checklists, and workflows) to produce implementation-ready GDD chapters in a consistent structure. The goal is to turn a game idea into a development-handoff document set, with clear specs and traceable assets.

## 🏗️ Agent First Game Dev Stack

This generator is the **Planning Layer** of our complete **Agent First Game Development Stack**:

1.  **OpenGDD Template**: The standardized protocol for AI game design ([opengdd.borninsea.com](https://opengdd.borninsea.com/)).
2.  **GDD Generator** (This Repo): The specialized agent team that turns ideas into specs.
3.  **BMAD Game Dev**: The engineering agents that build the game from these specs (Supports clients: Phaser, Godot, Unity, Cocos, Unreal; Supports AI envs: Cursor, Claude Code, OpenAI Codex, etc.).

See [Agent First Game Dev](https://opengdd.borninsea.com/opengdd-bmad-game-dev) for the full vision.
